# LOT WORKSPACE

## About
The LOT Workspace (Student Org) project aims to create a centralized online workspace tailored specifically for student 
organizations at Lovely Professional University (LPU). This platform will serve as a comprehensive solution for 
student groups to connect, collaborate, organize their activities, and manage their resources effectively.

## Key Features
### USER REGISTRATION AND ROLE MANAGEMENT
* Allow students to register for roles within their respective student organizations.
* Manage user roles and permissions to facilitate effective collaboration and organization management.
### EVENT MANAGEMENT
* Enable student organizations to post details of past, current, and upcoming events.
* Include features such as event posters, descriptions, registration forms for volunteers, and ticket purchasing options for guests.
### COMMUNICATION TOOLS
* Integrate live chat functionality to facilitate real-time communication among student organization members.
* Provide discussion forums or message boards for sharing ideas, updates, and announcements.
### RESOURCE SHARING
* Allow student organizations to share resources such as documents, templates, and guidelines.
* Enable members to access and contribute to a centralized repository of organizational materials.
### ADMIN DASHBOARD
* Provide an administrative dashboard for managing the platform and overseeing student organization activities.
* Include functionalities such as posting vacancies, managing event details, deleting users, marking attendance, and generating reports.
### ADVERTISING PLATFORM
* Offer student organizations the ability to advertise their events and activities to the wider LPU community.
* Allow guest users to purchase tickets for events directly through the platform.

## NATURE OF PROJECT
The project involves the development of a web-based platform using modern technologies and frameworks to facilitate communication and collaboration among LOT members. It requires expertise in web development, database management, and user interface design.

## NOVELTY OF THIS PROJECT

The project introduces a centralized platform tailored specifically for LOT members to streamline their organizational activities. While similar platforms exist, this project will focus on the unique needs and dynamics of LOT, offering customized features and functionalities.

## FEASIBILITY

It is feasible to carry out the proposed work with the facilities available in-house. The project will leverage existing resources such as development tools, programming languages, and server infrastructure available at Lovely Professional University.

## RESEARCH GAP
Through a survey conducted among LOT members, we observed the basic needs and problems they face while working with LOT. The survey results highlighted challenges such as communication barriers, lack of centralized information, and difficulty in advertising events. By researching and analysing these findings, we identified how a web-based workspace platform could address these challenges and reduce the problems faced by LOT members.
